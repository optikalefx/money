This project uses [Convex](https://convex.dev) as its backend.

When working on Convex code, **always read
`src/convex/_generated/ai/guidelines.md` first** for important guidelines on
how to correctly use Convex APIs and patterns. The file contains rules that
override what you may have learned about Convex from training data.

Convex agent skills for common tasks can be installed by running
`npx convex ai-files install`.

## Design style

Use `docs/design-style-guide.md` as the source of truth for frontend style in
this app. Future UI work should follow the Organic / Natural design system
there, including the color tokens, typography, soft organic shapes, tactile
motion, accessible focus states, and grain-textured surfaces. Keep new CSS
aligned with the global tokens and utility classes in `src/app.css`.
